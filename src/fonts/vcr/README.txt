SUPER-VCR MONO by ASmolBoy

A pixelated font based on the on-screen displays of VHS players/VCR's.

Font size of 12px (or multiple of) is recommended to get nice square pixels. No antialising.

Includes support for accented, currency, and other miscellaneous characters.
